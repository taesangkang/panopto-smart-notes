# Icon Files Required

The extension requires three icon files:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Quick Solution

You can create simple placeholder icons:

1. **Option 1**: Use any image editor to create simple PNG files with the required dimensions
   - Use a simple design (e.g., "PN" text or a note icon)
   - Save as PNG format

2. **Option 2**: Use an online icon generator
   - Visit https://www.favicon-generator.org/ or similar
   - Upload a simple image and generate icons

3. **Option 3**: Temporarily remove icon references
   - Edit `manifest.json` and remove the `"icons"` section
   - The extension will still work, just without a custom icon

## Recommended Design

For Panopto Smart Notes, consider:
- A notebook or document icon
- "PN" or "PSN" initials
- A combination of video + notes symbols

The icons should be simple and recognizable at small sizes.
